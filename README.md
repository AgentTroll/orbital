# `orbital`

# Build

Requires CMake and a C compiler to build. Requires LibSDL2
to be installed on your system (`sudo apt-get install libsdl2-dev` 
on Debian) to run.

``` shell
git clone https://github.com/AgentTroll/orbital.git
cd orbital
mkdir build && cd build
cmake .. && make
./orbital
```
